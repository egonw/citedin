Cited is available through an Application Programming Interface (API).

Return all included resource names

  • http://www.citedin.org/webservice.php/resources

Find all citations in all resources of a given Pubmed Identifier (i.e. 15489334)

  • http://www.citedin.org/webservice.php/citations?pmid=15489334

Find all citations in a given set of resources resources of a given Pubmed Identifier (i.e. 15489334)

  • http://www.citedin.org/webservice.php/citations?pmid=15489334&resource=Uniprot,Wikipathways
  • http://www.citedin.org/webservice.php/citations?pmid=15489334&resource=Uniprot

JSON

In these examples the output will be in an XML format. To obtain JSON output add "&format=json", i.e.: http://www.citedin.org/webservice.php/citations?pmid=15489334&format=json
